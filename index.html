📄 API DOC

Como integrar o sistema de keys no seu app

BASE URL

https://rbxpainelkeylol.netlify.app/api/keys

Todas as rotas usam esta URL base com o parâmetro ?action=

POST VALIDAR KEY

/api/keys?action=validate

Valida uma key e opcionalmente vincula ao HWID do dispositivo. Use no app C# para liberar acesso.

CAMPOTIPODESCRIÇÃO
keystringCódigo da key (ex: RBXF-XXXXX-XXXXX-XXXXX)
hwidstring?Hardware ID do dispositivo (opcional)
{
  "valid": true,
  "type": "free",
  "duration": "1d",
  "expiresAt": "2025-07-01T00:00:00.000Z",
  "daysLeft": 1
}

Exemplo C#:

var payload = new { key = keyInput, hwid = GetHWID() };
var json = JsonConvert.SerializeObject(payload);
var res = await _http.PostAsync(
  "https://rbxpainelkeylol.netlify.app/api/keys?action=validate",
  new StringContent(json, Encoding.UTF8, "application/json")
);
var data = JsonConvert.DeserializeObject<dynamic>(await res.Content.ReadAsStringAsync());
if ((bool)data.valid) { /* libera acesso */ }

POST CRIAR KEY

/api/keys?action=create

Cria uma nova key. Requer autenticação Basic (admin).

Headers: Authorization: Basic base64(email:senha)
Body: { "type": "free", "duration": "1d" }

Tipos: free · premium · dev

Durações: 1d · 30d · 90d · perm (dev só aceita perm)

GET LISTAR KEYS

/api/keys?action=list

Retorna stats + lista de todas as keys. Requer autenticação Basic.

POST RENOVAR KEY

/api/keys?action=renew
Body: { "code": "RBXF-XXXXX-XXXXX-XXXXX", "duration": "30d" }

POST REVOGAR / DELETAR

/api/keys?action=toggle
/api/keys?action=delete
Body: { "code": "RBXF-XXXXX-XXXXX-XXXXX" }
